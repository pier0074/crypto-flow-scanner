Parameter Configuration

Configure trading parameters per symbol and timeframe

Use the CLI tool to manage parameters:

# View all parameter overrides
python scripts/manage_parameters.py --list-all
# Set parameters for BTC/USDT on 1m
python scripts/manage_parameters.py --symbol BTC/USDT --timeframe 1m \
  --set-risk '{"max_risk_percent": 0.5}'
# View examples
python scripts/manage_parameters.py --examples
